Comece a Criar em 3D

Formas 3D

cube

Cria um bloco tridimensional regular (cubo) ou retangular.

cube: size = 30px position = center color = #38bdf8
sphere

Cria uma esfera perfeita no espaço 3D tridimensional.

sphere: size = 40px position = 0px 50px 0px color = red
pyramid-4

Cria uma pirâmide tridimensional com base quadrada perfeita.

pyramid-4: size = 30px 50px 30px position = -50px 0px 0px color = #eab308
torus

Cria um objeto tridimensional em formato de rosquinha vertical.

torus: size = 50px 50px 20px position = 50px 0px 10px color = purple
line

Cria uma linha reta conectando dois pontos no espaço vetorial 3D.

line: position = 10px 0px 20px, 0px 10px -20px thickness = standard color = #334155
text

Renderiza um objeto de texto com extrusão tridimensional real.

text = "Olá!": size = 20px position = 0px -60px 0px color = blue

Propriedades e Modificadores

size

Modifica a escala do objeto nos eixos X, Y e Z. Aceita 1 valor global, 3 específicos ou o termo padrão.

>> Global / Tridimensional / Padrão size = 30px size = 10px 20px 30px size = standard
position

Posicionamento cartesiano 3D (X = Direita, Y = Cima, Z = Frente). Linhas usam par de coordenadas separadas por vírgula.

>> Objeto / Centro / Linhas position = 10px 20px 30px position = center position = 10px 0px 20px, 0px 10px -20px
color

Aplica cor ao material do objeto. Suporta palavras reservadas diretas ou códigos hexadecimais.

color = red color = #ff0000
thickness

Define a espessura de linhas ou a volumetria de extrusão de blocos de texto tridimensionais.

thickness = 20px thickness = standard
>> (Comentário)

Linha ignorada pelo processamento sintático do motor do transpilador. Serve para anotações organizacionais.
