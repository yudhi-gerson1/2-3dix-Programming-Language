As nossas palavras-chaves já existentes:

Forma 3D

cube

Cria um cubo ou paralelepípedo tridimensional regular no espaço que pode ser totalmente customizado por tamanho, cor e posição.

cube:
  size = 40px
  position = center
  color = red
Forma 3D

sphere

Gera uma esfera (bola tridimensional) perfeita. A propriedade de tamanho define o diâmetro total da esfera.

sphere:
  size = 30px
  position = 0px 40px 0px
  color = #ffbb00
Forma 3D

pyramid-4

Modela uma pirâmide tridimensional clássica construída com uma base perfeitamente quadrada de 4 lados.

pyramid-4:
  size = standard
  position = center
  color = yellow
Forma 3D

torus

Cria uma rosquinha geométrica tridimensional renderizada "em pé" verticalmente no cenário.

torus:
  size = 30px
  position = center
  color = blue
Forma Especial

line (Linha 3D Vetorial)

Cria uma linha tridimensional física. Atenção: Você deve obrigatoriamente fornecer duas coordenadas completas na propriedade position separadas por vírgula para marcar o ponto inicial e o ponto final (Ponto A, Ponto B). Ela suporta estritamente as propriedades: position, color e thickness.

line:
  position = -120px 50px 0px, 120px 50px 0px
  color = #ff00ff
  thickness = 16px
Texto 3D

text (Texto Volumétrico)

Gera um texto físico tridimensional com profundidade extrudada. Para o compilador ler sem erros, o conteúdo textual deve ser atribuído obrigatoriamente com o sinal de igual (=) e entre aspas na mesma linha da palavra-chave. Suporta: size, position, color e thickness.

text = "YUDHI":
  size = 40px
  position = center
  color = red
  thickness = standard
Propriedade

size

Controla a escala do objeto. Aceita 1 valor (para eixos iguais), 3 valores específicos separados por espaço (Largura, Altura, Comprimento) ou a palavra-chave standard (50px para todos).

size = 30px
size = 10px 20px 30px
size = standard
Propriedade

position

Posiciona o objeto no espaço X (Esquerda/Direita), Y (Baixo/Cima) e Z (Trás/Frente). Valores positivos movem X para a direita, Y para cima e Z para a frente. Exige 3 dimensões obrigatórias, com exceção da palavra center (0px 0px 0px).

position = 10px 20px 30px
position = center
Propriedade

color

Define a cor da superfície do objeto. Aceita nomes em inglês ou valores hexadecimais.

color = red
color = #ff0000
Propriedade

thickness

Propriedade exclusiva para os elementos line e text. Controla a grossura da linha ou a profundidade da extrusão de volume do texto. O modo standard aplica 12px automáticos.

thickness = 20px
thickness = standard
Sintaxe

>> (Comentários)

Linhas iniciadas com duas setas são completamente ignoradas pelo motor gráfico, servindo para documentar seu código.

O que adicionamos na v1.1:

Banco de Dados

varSelector

Escopo global para armazenar e declarar variáveis. Aceita strings, números, pixels e a execução direta de expressões matemáticas com operadores aritméticos comuns.

varSelector:
  Pixel = 30px
  raio = 10px
  diametro = raio * 2

cube:
  size = 40px 50px Pixel
Atributo

id = "meu_objeto"

Identidade exclusiva para um único objeto tridimensional, garantindo a maior prioridade de herança e estilo local.

cube (id = "obj"):
  size = 50px
Atributo

class = "meus_objetos"

Categoriza e agrupa um ou vários objetos sob um mesmo rótulo para receber propriedades em lote.

sphere (class = "personagem"):
  color = #FFC080

cube (class = "personagem"):
  position = 0px -30px 0px
  color = red
Seletor

Seletor de Classe Puro

Permite declarar um bloco de estilização direto para uma classe. Os objetos vinculados herdarão essas configurações automaticamente.

obj:
  color = red

torus (class = "obj")
Nova Forma 3D

cylinder

Novo objeto nativo da v1.1. Modela um cilindro redondo volumétrico baseado na extrusão perfeita de um círculo.

cylinder:
  color = black
  size = 20px 50px
Nova Forma 3D

n-prism

Gera um prisma em pé regular com base poligonal de 'n' lados iguais. Substitua o prefixo 'n' pelo número do polígono desejado (ex: 3-prism cria um prisma de base triangular).

3-prism:
  color = blue
  size = 30px 50px 60px
Terminal

print ->

Imprime textos, números ou o valor resolvido de variáveis diretamente no console de saída. Possui um extrator automático que remove estritamente a unidade px dos números ao processar a saída.

print -> "Vamos aplicar comprimento em v1.1 de 2&3dix!"
print -> 3px

Saída limpa esperada no Console:

Vamos aplicar comprimento em v1.1 de 2&3dix!
3

Os três pilares em 3D:

Pilar 1

Lazy Definition (Definição Preguiçosa)

Ocorre quando você declara os objetos tridimensionais e suas atribuições de classe primeiro, mas a folha/bloco de estilo que configura essa classe é escrita mais abaixo no script.

cube (class = "blue")
cylinder (class = "blue")

blue:
  color = blue
Pilar 2

Inheritance (Herança)

O modelo tradicional de cascata: você define detalhadamente as propriedades estruturais de uma classe primeiro e, em seguida, cria os objetos abaixo herdando essas propriedades prontas.

blue:
  color = blue

cube (class = "blue")
sphere (class = "blue")
Pilar 3

Filho Independente (Independent Child)

Permite o ganho de herança global através da classe, mas concede total liberdade ao objeto de abrir um escopo local próprio para redefinir ou adicionar propriedades exclusivas.

blue:
  color = blue

cube (class = "blue"):
  size = 20px 40px 60px

pyramid-4 (class = "blue")
